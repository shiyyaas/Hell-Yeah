## Getting Started ğŸš€

### Prerequisites

- Git installed on your machine
- A Firebase account and configurations

### Installation

1. Clone this repository

```bash
git clone https://github.com/Jhayblaz/Hell-Yeah.git
cd Hell-Yeah
```

2. Edit the index.html

3. Set up Firebase
   - Go to [Firebase Console](https://console.firebase.google.com/)
   - Create a new project
   - Enable Firestore Database and Storage
   - Get your Firebase configuration object

4. Add your Firebase configuration
   - Paste your Firebase config in the frontend configuration file
   - That's it! The backend is now connected

5. Run the application

```bash
Double click HTML or Deploy with vercel
```
